# Project Template Maneger

## Dependencias
Antes de intalar PTM certifique-se de que o **curl wget e unzip** esteja instalado em sua maquina.

## Install
```bash
curl -o- https://raw.githubusercontent.com/RENATOADORNO/ptm/main/install.sh | bash
```

## Use
```bash
ptm
```

## Update
```bash
ptm-up
```
